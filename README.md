# RINT Templates Repository

Репозиторий шаблонов артефактов для платформы RINT.

## Структура

```
{Template.name}/
├── {Template.id}-v1.0.0.yaml
├── {Template.id}-v1.1.0.yaml
└── {Template.id}-v2.0.0.yaml
```

## Ветки

| Ветка   | Статус | Описание                                |
|---------|--------|-----------------------------------------|
| `draft` | DRAFT  | Черновики шаблонов, в разработке        |
| `main`  | ACTIVE | Активные шаблоны, готовы к использованию |

## Git Flow

1. Создание/редактирование шаблона → коммит в `draft`
2. Готовность к релизу → PR из `draft` в `main`
3. После merge → шаблоны становятся ACTIVE и доступны для использования в интеграциях
